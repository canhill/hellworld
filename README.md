- 💡 **简单易懂**：从零开始讲解 GitHub 使用流程  
- 🧭 **循序渐进**：涵盖克隆、提交、推送、分支等常用操作  
- 🛠️ **实用工具推荐**：提供图形界面客户端及插件建议  

---

## 📘 内容概览

### 1. 注册账号与配置 SSH 密钥
访问 [https://github.com](https://github.com) 创建账户，并根据官方教程配置本地环境。

### 2. 新建仓库并初始化
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin git@github.com:username/repo.git
git push -u origin main
```

### 3. 日常工作流模拟
| 操作 | 描述 |
|------|------|
| `git clone` | 下载远程仓库到本地 |
| `git status` | 查看修改状态 |
| `git branch` | 分支管理 |

### 4. Pull Request 协作方式演示
fork → 修改 → pull request → review merge
